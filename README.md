# Cplusplus-Program
The task was to build an item tracking program for the Corner Grocer company. They needed a program that analyzes the text records they generate throughout the day. These records list items purchased in chronological order from the time the store opens to the time it closes. They needed to know how often items are purchased so they can create the most effective layout for their customers and maximize efficiency and sales.  

What I did particularly well on this project was that I was able to use C++ map for efficient counting and retrieval, using a map<string, int> was an effective design decision because it automatically handled counting and sorting, making the code efficient and concise.  

There are several areas where the program could improve, for instance the searches are case sensitive. Converting everything to a lowercase would improve usability and accuracy.  

The most challenging part of my code was using map<string, int> effectively, it required me to understand how incrementing work and automatic key creation. I utilize compiler error messages, debugging step by step, and testing with simplified codes as tools to overcome certain challenges like resolving compiler and linker errors. I will continue to build my support network by practicing structured debugging and engaging in peer discussion and code review.  

 

Skills from this project that I find transferable are the file input and output handling, using STL containers like map, debugging compiler and linker errors. These skills will apply to my future coursework involving program design.  

I was able to make this program maintainable and readable by consistent indentation and formatting, using inline comments explaining the functionality of my code, and using a structured menu system. Adaptability in my program came from how I structure it. I put most of the logic inside the ItemTracker class instead of writing everything inside main(). This helped keep things organized and makes it easier to update later without breaking the entire program.
